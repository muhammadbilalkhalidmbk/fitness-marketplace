# 🛒 Fitness Marketplace

A full-stack **fitness e-commerce web application** built with **Python and Flask**, developed as a 2nd semester ISE (Introduction to Software Engineering) project at Air University.

---

## ✨ Features

- 🏋️ **Product Listings** — Browse fitness equipment and supplements
- 🔐 **User Authentication** — Register and login system
- 🛒 **Shopping Cart** — Add and manage items before checkout
- 🧑‍💼 **Seller Dashboard** — Sellers can list and manage their products
- 📦 **Order Management** — Track and manage orders
- 🖼️ **Image Upload** — Product images stored and displayed dynamically

---

## 🛠️ Tech Stack

- **Backend:** Python, Flask
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite / File-based storage
- **IDE:** VS Code / Replit

## ⚙️ How to Run

1. Clone the repository:
```bash
   git clone https://github.com/muhammadbilalkhalidmbk/fitness-marketplace.git
```
2. Install dependencies:
```bash
   pip install flask
```
3. Run the app:
```bash
   python main.py
```
4. Open your browser and go to:
5. http://localhost:5000
